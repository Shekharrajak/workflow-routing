workflow-routing
================

Default workflow routing for Meteor apps, including accounts-entry, static pages, and error pages.  


####  Installation  

````sh
# Should be as simple as cloning the repository...  
git clone https://github.com/awatson1978/workflow-routing.git

# And then running it...
cd workflow-routing
meteor
````
